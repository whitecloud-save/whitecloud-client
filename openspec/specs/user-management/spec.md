## 新增需求

### 需求：会员中心信息展示
系统必须在用户设置页面展示会员中心信息，包括 VIP 状态、会员到期时间和云存储空间使用情况。

#### 场景：未登录用户查看会员中心
- **当** 用户未登录
- **那么** 显示提示信息"登录账号以开启云同步存档功能"

#### 场景：付费会员查看会员中心
- **当** 用户已登录且 VIP 等级 > 0
- **那么** 显示 VIP 标识和会员到期时间

#### 场景：非付费会员查看会员中心
- **当** 用户已登录且 VIP 等级 = 0
- **那么** 显示成为会员按钮（按钮功能暂不实现）

#### 场景：所有登录用户查看云存储空间
- **当** 用户已登录
- **那么** 显示云存储空间使用情况，包括最大存储空间和已使用空间，使用进度条形式展示

### 需求：VIP 信息存储
系统必须在 UserService 中存储用户的 VIP 信息，包括 VIP 等级、存储空间和到期时间。

#### 场景：登录成功后存储 VIP 信息
- **当** 用户登录成功
- **那么** 从服务器返回的数据中提取并存储 VIP 信息

#### 场景：获取用户信息后更新 VIP 信息
- **当** 调用 fetchUserInfo 方法
- **那么** 从服务器返回的数据中提取并更新 VIP 信息

### 需求：云存储空间使用情况
系统必须提供云存储空间使用情况的实时显示。

#### 场景：显示存储空间使用情况
- **当** 用户登录成功
- **那么** 显示总存储空间和已使用空间，使用进度条可视化

#### 场景：VIP 等级与存储空间对应
- **当** 服务器返回 VIP 信息
- **那么** 根据 VIP 等级和已使用空间计算存储使用率
